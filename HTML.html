HTML

HyperText Markup Language의 약어로 웹 페이지를 위한 지배적인 마크업 언어이다.
HTML은 제목, 단락, 목록 등과 같은 본문을 위한 구조적 의미를 나타내는 것뿐만 아니라 링크, 인용과 그 밖의 항목으로 구조적 문서를 만들 수 있는 방법을 제공한다.

HTML 태그 사용법

  • <태그이름 옵션(또는 속성)="옵션값">내용</태그이름>

    • 태그의 종류

    1. 글씨 관련 태그

  • <strong>
    • 해당 콘텐츠의 중요성이나 심각함, 긴급함 등을 강조할 때 사용하는 태그
  • <b>
    • bold 약자로 글자를 굵게 만드는 태그
  • <i>
    • italic 약자로 글자를 기울게 만드는 태그
  • <u>
    • underline 약자로 글자에 밑줄을 만드는 태그
  • <s>
    • 취소선 태그로, 글자에 선을 긋는 태그
  • <em>
    • 강조 태그, i 태그와 비슷하지만 브라우저에서 음성지원 시 음성 강조 효과가 있다.
    <strong> 태그와 <b> 태그의 차이
    <b> 태그는 콘텐츠의 중요성보다는 텍스트 자체에 주의를 끌기 위해 사용되지만, <strong> 요소는 콘텐츠 자체의 중요성을 강조할 때 사용됩니다.

    2. 글자 관련 태그

  • <h1>, <h2>, <h3>, <h4>, <h5>, <h6>
    • h의 숫자가 커지면 글씨 크기가 작아진다.

  • <ruby>
    • 동아시아 문자의 발음을 표기할 때 사용하는 태그
  • <rt>
    • 발음, 번역 등을 나타내는 텍스트 부분을 지정하는 태그
  • <p>
    • 단락 태그

    3. 파일 관련 태그

  • <img>
    • 이미지를 넣는 태그
  • <video>
    • 비디오를 넣는 태그
  • <iframe>
    • 유튜브 영상을 가져올 때 주로 사용하는 태그
  • <audio>
    • 오디오를 넣는 태그

    4. 이동 관련 태그

  • <a>
    • 지정해둔 문서나 url로 이동하는 태그

    5. 리스트 관련 태그

  • <ol>
    • 번호가 붙는 리스트 태그
  • <ul>
    • 번호가 붙지 않는 리스트 태그
  • <li>
    • <ol> 태그나 <ul> 태그 안에 사용하는 리스트 태그

    6. 표 관련 태그

  • <table>
    • 표를 생성하는 태그이다. <tr> 태그와 <td> 태그와 <th> 태그를 같이 사용한다.
  • <tr>
    • 새로운 열을 생성한다.
  • <td>
    • 새로운 행을 생성한다.
  • <th>
    • 표의 이름이나 제목을 지정한다. 자동 가운데 정렬

    7. 구역 관련 태그

  • <div>
    • 영역을 지정하는 태그
  • <span>
    • in-line 형식으로 영역을 지정하는 태그
  • <header>
    • 보통 웹 상단에 웹서비스의 로고, 메뉴등이 위치하는 구역 태그
  • <footer>
    • 보통 웹 하단에 회사의 정보등이 위치하는 구역 태그
  • <nav>
    • 보통 메뉴를 위치하는 구역 태그
  • <section>
    • 보통 서브제목과 관련된 내용을 표시하는 구역 태그
  • <article>
    • 보통 신문기사나 블로그에 글을 작성할때 사용되는 구역 태그
